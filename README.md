# ShailKavach

A rockfall prediction and monitoring system with AI/ML capabilities.

**IMPORTANT: After cloning this repository, you MUST install dependencies before running the project.**

### Prerequisites
- Node.js (v16 or higher) - [Install Node.js](https://nodejs.org/) or [use nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- npm (comes with Node.js)

### Setup Instructions

```sh
# Step 1: Clone the repository
git clone https://github.com/iamharshitraj/ShailKavach.git

# Step 2: Navigate to the project directory
cd ShailKavach

# Step 3: Install dependencies (REQUIRED - this fixes "vite: command not found" error)
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:5173` (or the port shown in the terminal).

### Common Issues

**Error: "vite: command not found"**
- **Solution**: Run `npm install` first. The `node_modules` folder is not included in the repository and must be installed locally.
